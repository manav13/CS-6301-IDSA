How to compile and run the code for CuckooHashing: 
 
1. Unzip the file.
2. Open terminal inside directory kxs200019
3. Create .class file using command,
	javac CuckooHashing.java
	javac CuckooHashingDriver.java
	javac CuckooHashingTestDriver.java
4. Go to parent folder of kxs200019 using command cd ../
5. Run the class file using command,
	java kxs200019.CuckooHashingTestDriver sk-t01.txt
6. Make sure to import all the RBT test cases from the previous LP to use them here.

Open the java file in an IDE to add own test cases inside the main function(CuckooHashing.java).
